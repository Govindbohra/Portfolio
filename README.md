# Govind Bohra - Senior UI/UX Designer Portfolio

A clean, editorial-style single-page portfolio website showcasing senior-level UX design expertise and experience.

## Features

- **Clean Editorial Design**: Warm white background (#FAF8F5) with amber accent color (#E8693A)
- **Professional Typography**: Playfair Display for headings, DM Sans for body text
- **Smooth Animations**: Fade-in effects on scroll, hover transitions, smooth scrolling
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Fixed navbar with blur effect, timeline work experience, skill tags

## Sections

1. **Navbar**: Fixed navigation with logo and smooth scroll links
2. **Hero**: Large heading with professional tagline and CTA buttons
3. **About**: Personal bio with key statistics in large numbers
4. **Work Experience**: Timeline-style layout with detailed job descriptions
5. **Case Studies**: Grid of 4 project cards with hover effects
6. **Skills**: Organized skill tags grouped by category
7. **Contact**: Centered layout with email and LinkedIn buttons
8. **Footer**: Professional copyright notice

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Backdrop-filter)
- Vanilla JavaScript (Intersection Observer for scroll animations)
- Google Fonts (Playfair Display, DM Sans)

## Getting Started

1. Open `index.html` in any modern web browser
2. The portfolio will load with all animations and interactions working

## Customization

### Personal Information
- Update work experience details in the timeline section
- Replace case study placeholders with real project images
- Modify contact email and LinkedIn links
- Adjust colors in the CSS variables (search for #E8693A, #FAF8F5, #1A1A1A)

### Content Updates
- Edit the hero heading and subheading
- Update about section bio and statistics
- Add real case study descriptions and links
- Modify skill tags to match your expertise

### Styling
- Change fonts by updating the Google Fonts link
- Adjust spacing and layout in the CSS grid/flexbox properties
- Modify animation timings in the @keyframes sections
